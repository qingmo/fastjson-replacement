


## 使用方法
使用默认的`jackson`版本（当前`1.12.3`)
`maven`
```xml
<dependency>
    <groupId>com.github.qingmo</groupId>
    <artifact>fastjson-replacement</artifact>
    <version>$version</version>
</dependency>
```

使用自定义版本的`jackson`版本
